# BabyGoo Shop

Tienda online y plataforma de comercio electrónico de ropa para bebé. Esta aplicación web permite a los clientes explorar y comprar productos de forma intuitiva, ofreciendo opciones de envíos a todo México y Venezuela. Además, cuenta con un panel administrativo robusto para la gestión completa de la tienda.

## Tecnologías Utilizadas (Stack)

- **Frontend**: React + Vite
- **Interfaz de Usuario (UI)**: shadcn/ui + Tailwind CSS
- **Backend**: MedusaJS v2
- **Lenguaje**: TypeScript

## Características Principales

### 🛒 Tienda en Línea (Vista Cliente)
- Exploración intuitiva de un catálogo de ropa y accesorios para bebé.
- Sistema de carrito de compras dinámico y optimizado.
- Selección por región y moneda (MXN/VES).
- Visualización de banners promocionales y ofertas especiales.
- Diseño visual moderno, completamente responsivo (adaptable a dispositivos móviles y de escritorio).

![Vista del Cliente](public/images/Tienda%20-%20cliente.png)

---

### ⚙️ Panel de Administración (Dashboard Admin)
- Gestión del inventario y actualización del listado de productos con paginación y búsqueda.
- Configuración de ajustes generales y seguridad de la tienda.
- Control personalizable del banner promocional principal (gestión de las imágenes y enlaces del carrusel).
- Sistema de acceso controlado con protección de rutas e inicio de sesión.
